# 静波瑜伽公众号开发

## 安装的包

- express (服务器)
    
- sha1 (sha1加密)

- request和request-promise-native（服务器中发送请求，不可以使用Ajax）

- xml2js (将xml解析为JS对象)

- ejs (嵌入式JavaScript模板)

- mongoose (数据库)

## how to use

1.first install the packages

```
$npm install
```

2.then run app.js

```
right click the app.js and run it
```
